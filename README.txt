ПРОЕКТ: Мини-сайт для портфолио

Что внутри:
- index.html — основная одностраничная страница
- styles.css — стили сайта
- script.js — небольшая интерактивность
- sample_raw_catalog.csv — демонстрация сырого результата
- sample_clean_catalog.csv — демонстрация чистого результата
- sample_portfolio_demo.xlsx — Excel-файл примера

Как запустить:
1. Распаковать все файлы в одну папку
2. Загрузить содержимое папки в репозиторий GitHub без подпапок
3. Подключить репозиторий к Cloudflare Pages
4. Build command можно не указывать
5. Output directory — тоже не нужна, если Cloudflare берет корень проекта

Сайт полностью статический и готов для деплоя как есть.
